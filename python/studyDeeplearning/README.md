###2016/09/29

ちょうど面白そうな本があったのでやりはじめることに。ディープランニングを学んで何が得られるかは予想の難しいところだがプログラムを扱うならば、とにかく常に新しい技術に触れておかないとすぐに置いていかれてしまう気がするのでやるに越したことはないだろう。
